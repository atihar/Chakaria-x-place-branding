# Chakaria - x
## _A place branding project by mahir_

[![N|Solid](https://cldup.com/dTxpPi9lDf.thumb.png)](https://nodesource.com/products/nsolid)

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)

An online platform to give a glance of the chakaria to increase experience of the residence and tourists.

## Features

- Places to see in chakaria
- Places to stay in chakaria
- Important links and emergency service listed
- Geofencing
- Insights about the geographical placement

## Installation

Chakaria-x requires [Node.js](https://nodejs.org/) v16+ to run.

Install the dependencies and devDependencies and start the server.

```sh
cd dillinger
npm i
node app
```

For production environments...

```sh
npm install --production
NODE_ENV=production node app
```


## Docker

Chakaria-x is very easy to install and deploy in a Docker container.

By default, the Docker will expose port 8080, so change this within the
Dockerfile if necessary. When ready, simply use the Dockerfile to
build the image.

```sh
cd chakaria-x
docker build -t <youruser>/chakaria:${package.json.version} .
```

This will create the chakaria-x image and pull in the necessary dependencies.
Be sure to swap out `${package.json.version}` with the actual
version of Chakaria-x.

Once done, run the Docker image and map the port to whatever you wish on
your host. In this example, we simply map port 8000 of the host to
port 8080 of the Docker (or whatever port was exposed in the Dockerfile):

```sh
docker run -d -p 8000:8080 --restart=always --cap-add=SYS_ADMIN --name=Chakaria-x <youruser>/chakaria-x:${package.json.version}
```

> Note: `--capt-add=SYS-ADMIN` is required for PDF rendering.

Verify the deployment by navigating to your server address in
your preferred browser.

```sh
127.0.0.1:8000
```
